# Account Management (Mini Project 3)

Simple React app that lets you store and manage account information for different services (e.g., Google, Instagram, etc.).  

## Features
- Use of components (App, Account form, Card)
- Display accounts grouped by service (No card duplicates)
- Add a new account/edit account info/delete accounts
- Search/filter by services and accounts
- List View button to display all accounts in a simple UL/LI format (as per the project details)
